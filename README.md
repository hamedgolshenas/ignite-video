# Ignite Video

An Ignite plugin for react-native-video.

## Usage

1. 
```
$ ignite add video
```

2.
```
$ ignite add video MyVideo
```

## Contributing

1. Clone this repo
2. Run `npm install`
3. Run `npm test`
4. Check out a branch and make your changes
5. Write tests for those changes
6. Submit a pull request back upstream
